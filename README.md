# Gateway-Assignment


<h2>Description</h2>
In this project I will be will be researching, installing, and configuring an enterprise gateway for Power BI. I will also configure MS SQL Server as my on-premise source.
<br />


<h2>Plan of Action</h2>

- <b>Install MS SQL Server Management Studio</b> 
- <b>Install PowerBI Desktop</b>
- <b>Install On Premises Data Gateway(Personal Mode)</b>
- <b>Import a dataset</b>
- <b>Connect to On Premise data in SQL Server</b>

<h2>Environments Used </h2>

- <b>VirtualBox Windows 11</b> (21H2)
